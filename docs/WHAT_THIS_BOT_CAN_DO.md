# What This Bot Can Do

Please try `/help` first in the Telegram bot. That command gives the simplest in-product explanation of the bot's purpose, scope, and how to use it. The Telegram command menu can also expose `/start`, `/help`, and `/language`.

## English

**Purpose**

Delhi Scheme Saathi is a conversational guide for Delhi government welfare schemes. It helps a user describe their situation in plain language and then shows relevant schemes and next-step guidance.

**Current capabilities**

- Understand user requests in English, Hindi, or Hinglish
- Ask short follow-up questions such as age, category, income, gender, or situation
- Find relevant schemes from the current scheme catalog
- Explain one selected scheme in simpler language
- Answer follow-up questions about likely eligibility and why a scheme was suggested
- Show documents, application steps, and common rejection warnings
- Let the user switch the bot language with `/language` or the language buttons shown in help
- Let the user restart with `/start` or by saying `start over` or `restart`
- Let the user end the conversation by saying `bye` and start fresh again later

**How judges can test it quickly**

1. Open the Telegram bot and type `/help` first.
2. Optionally try `/language` to switch between English, Hindi, and Hinglish.
3. Then try one simple need such as:
   - `I need widow pension help`
   - `I need housing assistance`
   - `मुझे स्वास्थ्य सहायता चाहिए`
4. Answer the short profile questions.
5. Open one scheme and ask follow-ups like:
   - `Am I eligible?`
   - `Why did you suggest this scheme?`
   - `What documents are needed?`
   - `How do I apply?`

**Important**

This bot is a guidance tool, not the final government authority. Final eligibility and approval depend on official rules, document verification, and department review.

## हिंदी

**उद्देश्य**

दिल्ली स्कीम साथी दिल्ली सरकार की कल्याणकारी योजनाओं के लिए एक संवादात्मक गाइड है। यह उपयोगकर्ता की स्थिति को सरल भाषा में समझकर संबंधित योजनाएं और आगे की जानकारी बताता है।

**अभी यह बॉट क्या कर सकता है**

- हिंदी, अंग्रेज़ी, और हिंग्लिश में उपयोगकर्ता की बात समझना
- उम्र, श्रेणी, आय, लिंग, या स्थिति जैसी छोटी follow-up जानकारी पूछना
- मौजूदा scheme catalog से संबंधित योजनाएं ढूंढना
- चुनी गई योजना को सरल भाषा में समझाना
- संभावित पात्रता और यह योजना क्यों दिखाई गई, जैसे सवालों का जवाब देना
- दस्तावेज, आवेदन प्रक्रिया, और rejection warnings बताना
- `/language` या help में दिखने वाले language buttons से भाषा बदलने देना
- `/start` या `start over` / `restart` कहकर बातचीत फिर से शुरू करने देना
- `bye` कहकर बातचीत समाप्त करने देना, ताकि बाद में नई शुरुआत की जा सके

**जज इसे जल्दी कैसे परख सकते हैं**

1. Telegram bot खोलें और सबसे पहले `/help` लिखें।
2. चाहें तो `/language` से भाषा बदलकर देखें।
3. फिर एक सरल जरूरत लिखें, जैसे:
   - `मुझे विधवा पेंशन की जानकारी चाहिए`
   - `मुझे घर के लिए योजना चाहिए`
   - `I need education loan help`
4. बॉट के छोटे profile सवालों का जवाब दें।
5. किसी एक योजना को खोलकर आगे पूछें:
   - `क्या मैं पात्र हूं?`
   - `यह योजना क्यों दिखाई गई?`
   - `कौन-कौन से दस्तावेज चाहिए?`
   - `आवेदन कैसे करें?`

**महत्वपूर्ण**

यह बॉट मार्गदर्शन के लिए है। अंतिम पात्रता और मंजूरी सरकारी नियमों, दस्तावेज़ सत्यापन, और विभागीय जांच पर निर्भर करती है।
